Welcome to Prose

Prose is a very simple stylesheet you can use to quickly prepare some text for pretty online display. Prose is elegant, responsive and dead-simple. The document you're viewing right now uses Prose with all the default settings.

If you'd prefer a more technical explanation, Prose is a very small, easily customizable Sass file, designed to be used in conjunction with normalize.css to turn some quick HTML markup into a presentable web-page. The only dependency is a container div for a simple responsive design.

Example

Here's a basic HTML template to get you started using Prose.css:

<!DOCTYPE HTML>
<html lang="en">
    <head>
        <title>Prose.css</title>
        <meta http-equiv="content-type" content="text/html; charset=utf-8" />
        <meta name=viewport content="width=device-width, initial-scale=1">
        <link rel="stylesheet" href="css/normalize.css">
        <link rel="stylesheet" href="css/prose.css">
    </head>
    <body>
        <div id="container">
            <!-- Your markup goes here -->
        </div>
    </body>
</html>

Easily customize variables like the text-width, padding, colors, and so on, or don't touch a thing and it will still look great!
